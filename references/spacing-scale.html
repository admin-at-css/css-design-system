Exponential Spacing Scale

Following Refactoring UI principles, this spacing scale uses a 4px base unit with exponential growth. Smaller values are tightly packed for fine adjustments, while larger values grow quickly for layout spacing. This creates visual rhythm without overwhelming choice.
