Shadow & Elevation

Two-part shadow system with elevation levels and z-index stack

Two-Part Shadow Technique

Following Refactoring UI principles, our shadows use two layers: a larger, softer shadow for ambient light and a smaller, darker shadow for direct light. This creates more realistic depth perception than single-layer shadows.

Two-Part Shadow Anatomy

Direct light + ambient light = realistic depth

Direct Shadow (Sharper)

0 4px 6px -1px rgb(0 0 0 / 0.1)

Mimics light from above, creates the primary shadow shape.

Ambient Shadow (Softer)

0 2px 4px -2px rgb(0 0 0 / 0.1)

Diffuse environmental light, adds subtle depth.

Combined Result

0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1)
